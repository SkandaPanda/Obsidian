|Celltyp|Funktion|
|---|---|
|**Erytrocyter**|Transporterar syre (O2O_2O2​) från lungorna till vävnaderna och koldioxid (CO2CO_2CO2​) tillbaka till lungorna via hemoglobin. **Ingen roll i immunförsvaret.**|
|**Trombocyter**|Centrala för **hemostas**: bildar pluggar vid kärlskador och aktiverar koagulationskaskaden för att stoppa blödning. **Ingen direkt roll i immunförsvaret.**|

| Celltyp                     | Funktion                                                                                                                            | Uppdatering/Förtydligande                                                                                             |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| **Neutrofila granulocyter** | "Första insatsstyrkan": fagocyterar bakterier och andra patogener. **Rekryteras från blodkärl till infektionshärden via diapedes.** | **Diapedes** är processen där immunceller tar sig ut ur blodkärl till vävnaden. Neutrofiler är de första att anlända. |
| **Eosinofila granulocyter** | Specialiserade på att attackera **parasiter** och spelar en roll vid allergiska reaktioner.                                         | **Föreläsningen betonar deras roll mot parasiter och allergier.**                                                     |
| **Basofila granulocyter**   | Frisätter **histamin** och **heparin**, vilket bidrar till inflammation och allergiska reaktioner.                                  | **Histamin** ökar kärlpermeabilitet och stimulerar inflammation.                                                      |
| **Monocyter**               | Utvecklas till **makrofager**: fagocyterar patogener och döda celler, fungerar som **antigenpresenterande celler**.                 | **Stationära makrofager** finns redan i vävnaden, medan nya rekryteras från blodet.                                   |
| **NK-celler**               | Dödar celler som saknar **MHC I** (t.ex. virusinfekterade eller cancerceller) genom **apoptos**.                                    | **NK-celler är viktiga för att eliminera celler som virus har lurat att sluta uttrycka MHC I.**                       |

| Celltyp          | Funktion                                                                                                                    | Uppdatering/Förtydligande                                                                                                            |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| **B-lymfocyter** | Känner igen **specifika antigen** och differentierar till **plasmaceller** (producerar antikroppar) eller **minnesceller**. | **Klonal selektion**: Endast B-celler med rätt receptor aktiveras. **Slumpmässig rekombinering** skapar unik antikroppsspecificitet. |
| **T-lymfocyter** | **T-mördarceller**: Dödar virusinfekterade/cancerceller. **T-hjälparceller**: Dirigerar immunsvaret via **cytokiner**.      | **T-mördarceller** känner igen **icke-kroppsegna peptider** i **MHC I**. **T-hjälparceller** binder till **MHC II**.                 |
